The CSS Box Model

The CSS box model identifies the four parts of an HTML element that can be styled: content, padding, border, and margin.

The Four Parts of the CSS Box Model

  1. Content
  2. Padding
  3. Border
  4. Margin

Content

The content box holds the content, like text or images. Its size is adjustable with the CSS width and height properties.

Padding

The padding surrounds the content box, creating space between the content and the border.

Border

The border encloses the padding and content. It can be styled using properties like border-style, border-width, and border-color.

Margin

The margin creates space between the border and surrounding elements.
